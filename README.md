# KiCADscripts
Python scripts for KiCAD

# How to Use
- Download python script (xxx.py) for your local folder.
- Open Pcbnew(KiCAD)
- Click "Tools" -> "Scripting Console" in Pcbnew
- Input " execfile('\<folder pass\>/\<file name\>.py') " in Console
